README file for Commit with Voting use-case

1. Open remix.ethereum.org in chrome browser and create new file to proceed,
2. Copy the State Machine and Self Destruct code and paste there.
3. Make sure that the current version in the Compile button the upper right corner
 	is the same in stated in pragma solidity and start to compile it (Ctrl + S).
4. Go to the run button besides compile and check the Environment that needs to 
	be in JavaScript VM.
5. Click the Deploy button and there will be popping up right on the lower right
	corner and click to it.
6. Another popping up button will appear and for more informations, you can follow
	the instructions clearer in https://karl.tech/learning-solidity-part-2-voting/
